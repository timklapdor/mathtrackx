The following table summarises all the differentiation rules you’ve learnt in this course. It is essential that you remember all the fundamental rules. With these you should be able to tackle any problem involving differentiation.

The fundamentals
\(f(x)\) \(f’(x)\)
\(c\) constant \(0\)
\(x^n\) for a natural number\(n\)\(x^a\) for any real \(a\neq0\) \(nx^{n-1}\)\(ax^{a-1}\)
\(e^{x}\)\(a^{x}\) for any \(a>0\) \(e^{x}\)\(\ln(a)a^{x}\)
\(\log(x)\)\(\ln(x)\)\(log_{b}(x)\) \(1/(\ln(10)x)\)\(1/x\)\(1/(\ln(b)x)\)
\(\sin(x)\)\(\cos(x)\)\(\tan(x)\) \(\cos(x)\)\(-\sin(x)\)\(1/\cos(x)^{2}\)
\(a\,g(x)\) for a real number\(a\) \(a\,g’(x)\)
\(g(x)+h(x)\)\(g(x)-h(x)\) \(g’(x)+h’(x)\)\(g’(x)-h’(x)\)
\(g(x)\, h(x)\) \(g’(x)\,h(x)+g(x)\,h’(x)\)
\(g(x)/h(x)\) \((g’(x)\,h(x)-g(x)\,h’(x))/h(x)^2\)
\(g(h(x))\) \(h’(x)\,g’(h(x))\)

The following two tables of common chain rule examples can all be obtained using all the fundamental rules above. It is therefore not essential to remember all of these, but there are occasions when they may come in handy and save you some time.

Common chain rule examples 1
Function Derivative
\(f(x)^{a}\) for natural,integer, rational or real \(a\) \(a\, f’(x)\, f(x)^{a-1}\)
\(e^{f(x)}\) \(f’(x)\, e^{f(x)}\)
\(a^{f(x)}\) for any \(a>0\) \(\ln(a)\, f’(x)\, a^{f(x)}\)
\(\log(f(x))\) \(f’(x) / (\ln(10)\, f(x))\)
\(\ln(f(x))\) \(f’(x) / f(x)\)
\(log_{b}(f(x))\) \(f’(x) / (\ln(b)\,f(x))\)
\(\sin(f(x))\) \(f’(x)\, \cos(f(x))\)
\(\cos(f(x))\) \(-f’(x)\, \sin(f(x))\)
\(\tan(f(x))\) \(f’(x) / \cos(f(x))^{2}\)
Common chain rule examples 2
Function Derivative
\(f(x^a)\) for natural, integer,rational or real \(a\) \(a\, x^{a-1}\, f’(x^{a})\)
\(f\left(e^{x}\right)\) \(e^{x}\,f’\left(e^{x}\right)\)
\(f\left(a^{x}\right)\) for any\(a>0\) \(\ln(a)\,a^{x}\,f’\left(a^{x}\right)\)
\(f(\log(x)))\) \(f’(\log(x))/(\ln(10)x)\)
\(f(\ln(x))\) \(f’(\ln(x))/x\)
\(f\left(\log_{b}(x)\right)\) \(f’\left(log_{b}(x)\right)/(\ln(b)x)\)
\(f(\sin(x))\) \(\cos(x)\,f’(\sin(x))\)
\(f(\cos(x))\) \(-\sin(x)\,f’(\cos(x))\)
\(f(\tan(x))\) \(f’(\tan(x)) / \cos(x)^{2}\)

If you ever need to determine the equation of a tangent or normal line, then the following two rules are all you need.

Tangent and normal lines
Tangent of \(f(x)\) at \(a\) \(y=f’(a)(x-a)+f(a)\)
Normal of \(f(x)\) at \(a\) \(y=-(x-a)/f’(a)+f(a)\)
