Math in Discussion

During your study you may want to discuss mathematical expressions. To do this you'll need to provide to use MathJax to write these equations. The information below will help you to write these expressions so that they display correctly.

You can create formatted mathematical expressions in the course discussions of your edX course. The course discussions use MathJax, which uses a subset of Tex/LaTeX, to format the text that you type. You must surround your expression with dollar signs ($) for the expression to appear correctly. See the following table for common expressions used in courses.

NoteThe formatting that you use to enter mathematical expressions in the course discussions is different from the formatting that you use in problems in the courseware. For more information about formatting mathematical expressions in the courseware, see Entering Mathematical and Scientific Expressions in the edX Learner's Guide.

Using Dollar Signs in Course Discussions

Course discussions use dollar signs ($) to signal that text between two dollar signs should be formatted as a mathematical expression. This can make it tricky if you want to use the dollar sign as a currency symbol and not to indicate the start of a mathematical expression. To work around this problem, when you want to use more than one dollar sign in a course discussion, use this expression to represent the dollar sign: 

$\$$

For example, enter this:

The full tour costs $\$$20, while the self-guided tour costs $\$$8.

And the text displayed in the discussion will look like this:

The full tour costs $20, while the self-guided tour costs $8.
